# KCNA Certification Study Notes (EN)

This repository contains study notes and resources to help prepare for the **Kubernetes and Cloud Native Associate (KCNA)** certification exam.

## Contents

- **Course Notes**: A series of Markdown (`.md`) files in Italian that provide detailed coverage of all topics from the official KCNA course syllabus.
- **Anki Flashcards**: A CSV file (`kcna_flashcards.csv`) containing numerous flashcards. This file is formatted for easy import into the [Anki](https://apps.ankiweb.net/) spaced repetition software, helping you memorize key definitions, acronyms, and concepts.
